# spotify-analytics-dashboard

## Project Overview
End-to-end analytics project analyzing 97,980 Spotify 
tracks to uncover trends in music popularity, audio 
features, genres and artist performance.

## Business Questions
- Which genres have the highest average popularity?
- What audio features drive track popularity?
- Who are the top 10 artists by track count?
- How is popularity distributed across all tracks?

## Tools Used
| Tool | Purpose |
|------|---------|
| Python (Pandas) | Data cleaning & EDA |
| SQL (SQLite) | Data storage & querying |
| Matplotlib/Seaborn | Exploratory charts |
| Tableau Public | Final dashboard |

## Key Insights
- pop-film & k-pop are the most popular genres
- The Beatles have the most tracks (279)
- Average track popularity is 38.67
- Most tracks score between 20-50 popularity
